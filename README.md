自我总结，这次比赛能不能取得高名次，取决于以下几点：
---
1.本地训练测试集划分，是否符合赛题要求，同时还没有数据泄露

2.数据维度是否理解透彻，一些特别重要的维度，需要从实际的业务出发，做合理联想、尝试。一味单纯的统计维度很难超过契合业务逻辑的特征的效果

3.对评测的函数的理解，是否将其修正以直接优化评测目标

大体上，做到第一点外加基础数据统计，就基本可以top50，而剩下的，就看在2,3点上的完成度了
